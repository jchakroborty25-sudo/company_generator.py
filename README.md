# AI FP&A Copilot

## Overview

AI FP&A Copilot is an enterprise-style Finance Analytics project designed to automate financial reporting and provide AI-assisted business insights.

## Business Problem

Finance teams spend hours every month preparing:

- Budget vs Actual reports
- Variance analysis
- Executive commentary
- Management dashboards

This project aims to automate these activities using Python, Power BI, SQL, and AI.

## Technology Stack

- Python
- Power BI
- SQL
- Excel
- GitHub
- ChatGPT
- Microsoft Copilot
